
# Aurora Studio — デザイン比較用デモ

## 追加: Bug Panel を表示/非表示
- パネル右上の **「非表示」** ボタンで隠せます
- 右下に出る **「Bug Panel」** ボタンで再表示できます
- ショートカット: **P** キーで表示/非表示トグル（入力中は無効）

## 使い方
1. `index.html` をブラウザで開きます。
2. 右下の **Design Bug Panel** でチェックを入れると、各種デザイン不具合が注入されます。
3. 「ランダム」ボタンで複数の不具合をランダム適用できます。「リセット」で基準デザインに戻します。

## 注入できる不具合一覧
- **色ズレ**: ブランドカラー（青）を赤系に置換
- **フォント/ウェイト不一致**: セリフ体に変更、見出しのウェイト低下
- **余白縮小**: 余白・グリッド間隔を意図的に詰める
- **整列崩れ**: セクションの整列が左寄せに
- **低コントラスト**: テキストやボタンのコントラストを低下
- **グリッド崩壊**: カードやギャラリーが重なり、見出しと干渉
- **ヒーロー過拡大**: ヒーロー画像が拡大してはみ出し
- **画像欠落**: 指定のギャラリー画像を 404 に切り替え
- **z-index不具合**: ヘッダーの重なり順を異常化
- **レスポンシブ破壊**: メディアクエリを無効化してレイアウト破綻

## Figma 取り込み
- `design-spec-figma.svg` を Figma にドラッグ&ドロップしてください。
- ベクター要素として読み込まれるため、**基準デザイン**としての参照に使えます。
